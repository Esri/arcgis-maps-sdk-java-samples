# OpenStreetMap Layer

Display tiles from OpenStreetMap.

![](OpenStreetMapLayer.png)

## How it works
To create a map with an OpenStreetMap basemap:

   1. Create an `ArcGISMap` with a `Basemap.Type.OPEN_STREET_MAP`.
   2. Set the map to a `MapView`.


## Relevant API


   * ArcGISMap
   * Basemap
   * Basemap.Type
   * MapView
